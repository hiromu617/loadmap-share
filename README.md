# CCC-2021
ロードマップをシェアするアプリケーション

# dev

```bash
#イメージをビルド
$ docker-compose build --no-cache
#nodo_modulesをインストール
$ docker-compose run --rm front yarn
#起動
$ docker-compose up -d
```
 
# used liblary in front

- [tailwindcss](https://tailwindcss.com/)
- [SWR](https://swr.vercel.app/ja)
- [headless UI](https://headlessui.dev/)
